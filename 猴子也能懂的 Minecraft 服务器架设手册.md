# 猴子也能懂的 Minecraft 服务器架设手册 如何在一台空空如也的 linux 服务器中启动一个漂亮的 Minecraft Server

## 0. 为什么要从零开始启动服务器呢？ Muiticraft 它不香吗？

这是个好问题。他的一个答案是，如果你能弄到一个学生云主机，你的服务器月租金甚至可以低到 10 块钱左右（如果你并不需要很高的性能的话）。而如果你想要一个真正高性能的服务器来运行那些真正庞大的整合包，市面上常见的出租用 Muiticraft 实例往往不能满足你的性能需求。另外，淘宝上的 Muiticraft 服务器一般都会有不少版本和mod安装上的限制，难道你就不想拥有完全的自由吗？

### 0.1. 为什么要使用 linux 开服？我可以使用 Windows Server 吗？

原则上，你可以。Windows Server 有着漂亮的 GUI（图形用户界面，如果你从来没有见过这个缩写的话） ，你可以像操作你自己的电脑一样操作它。但一切命运的赠礼皆有代价——你需要耗费相当的内存来维持操作系统本身的运行，这意味着你能分配给服务器本身的资源会少不少。另外，使用 linux 开服，会让你看上去比较专业。如果你想问最关键的原因，那么答案是：你不一定随时都有 Windows Server 可用，但所有的服务器至少都能给你安装一个 Debain 或者 Cent OS （它们两个都是服务器用的 Linux 版本）系统。

## 1 连接到我的服务器

很显然，仅仅“拥有”一台服务器是不够的。你需要能够连接，登陆并操纵这台服务器。Windows 10 PowerShell 提供了十分方便的 SSH 连接工具，所以我们就使用它吧。如果你不幸没有 Win 10，你可以使用 Putty 代替，但它看上去没有那么帅。

### 1.1 SSH——这是什么？

SSH 是 安全外壳终端（Secure Shell）的缩写。简单的说，它允许你安全的远程登录一个网络上的 Linux 服务器——只要你有用户名，密码和IP地址。然后，你就可以像操作自己的电脑一样，使用 Shell 命令行操作远程服务器了。

欢迎回来。让我们开始吧。假设我们的服务器公网 IP 是 11.45.148.10。那么，在 PowerShell 里键入这个命令：

```shell script
ssh root@11.45.148.10
```

这个命令的意思是这样的：

```shell script
ssh　	root	@	11.45.148.10
远程连接	用户名	在 	主机IP
```

接下来，它会要求你输入密码。（这里需要多嘴一句，在 shell 中，请不要使用 ctrl+v 粘贴。你可以使用鼠标右键粘贴——等下，我刚才是不是告诉你要把鼠标扔掉来着？）你输入的密码是完全不可见的，不要因为屏幕上什么变化都没有而担心，输入完成后敲回车即可。接下来，你会看到界面变成这样（字体会整个刷的一下换掉）：
